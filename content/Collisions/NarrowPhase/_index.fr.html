Narrow Phase

Définition :

La narrow phase est la partie où les collisions entre les formes sont testées avec leur réelle géometrie.
Cela veut aussi dire que ces collisions sont précises, mais aussi lourdes en performance.

La façon brute de faire les choses serait de tester tous les triangles entre eux.
Cependant, cela serait trop lent.

Des algorithmes ont été créés pour rendre ça plus rapide.
La plupart d'entre eux utilisent les propertiés des formes convexes.

Conditions requises :

Pour que cette phase soit utile, elle doit :
  • Le point de collision
  • La normale de l'impact
  • La distance de pénétration entre les deux formes

Sans cela, on ne serait pas capable de faire une réponse de collision convenable après coup.
